# GitPortfolio

### Telegram-бот для демонстрации проектов GitHub

Бот помогает работодателям быстро находить релевантные проекты в вашем GitHub по выбранным технологиям.

![image](https://github.com/user-attachments/assets/fca26688-efe5-4f95-b592-a49db728a885)


## Установка

Клонируйте репозиторий:
```bash
git clone https://github.com/MariaOstanina/GitPortfolio
```
Установите необходимые зависимости:
```bash
npm install
```
Настройте бота:

Получите токен у @BotFather

Создайте файл .env:
BOT_TOKEN=ваш_токен  

## Запуск

```bash
node index.js  
```
